Поиск ведется по текстовым файлам-документам, путь к которым указан в "config.json" в разделе "files".
Формат файла "config.json":
"name"    - название проекта
"version" - версия файла конфигурации "config.json".
            Должна совпадать с версией проекта.
"max_responses"   - максимальное количество ответов в выдаче
"update_interval" - интервал обновления индекса в минутах
"files"   - список документов, по которым будет вестись поиск

Ищутся поисковые фразы, указанные в файле "requests.json".

Результаты поиска сохраняются в файл "answer.json"
Формат файла "answer.json":
"reqID"   - идентификатор запроса 
"request" - строка-запрос, 
"reqDone" - флаг "поиск произведен", 
"result"  - флаг "результат поиска положительный", 
"relevance" - относительная релевантность результатов поиска. 
              Указываются идентификаторы документов в порядке убывания релевантности.
"docIDs"    - список идентификаторов документов

Перед началом поиска необходимо проиндексировать документы - построить индекс.
